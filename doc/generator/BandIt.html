Class BandIt

BandIt is a library above RaphaelJS to create workflows in HTML/JS.

Constructor

BandIt ( diveditor , width , height )
Parameters:
diveditor <object> Name of the div where editor will be put
width <object> Width of the div
height <object> Height of the div

Properties

mode - object

Operation mode: 0 : drag and drop, selection 1 : link 2 : delete 3 : group
